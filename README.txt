Capybara Clicker 
======================

Description:
Capybara Clicker is a browser-based clicker game where you earn points by clicking on a capybara image. The game features upgrades that increase Capys Per Click (CPC) and Capys Per Second (CPS). Progress is saved using local storage, allowing you to continue from where you left off.

How to Play:  
1. Click on the capybara to earn points.
2. Buy upgrades in the shop to increase your CPC and CPS.
3. Keep clicking and upgrading to grow your score.

Features:  
- CPC (Capys per Click): Earn more points with each click.
- CPS (Capys per Second): Automatically earn points over time.
- Shop system with CPC and CPS upgrades.
- Reset button to start over at any time.
- Settings menu with a sound toggle.
- Local storage saves your score, CPC, CPS, and sound preferences across sessions.
- Diagonally scrolling blue background for visual appeal.

Files:  
1. **index.html** - Main HTML file for the game structure.
2. **styles.css** - Contains the game's layout and styling.
3. **script.js** - Handles game logic including clicks, upgrades, local storage, and settings.

Installation:  
1. Download the `index.html`, `styles.css`, `script.js`, and asset folders (Graphics, Sounds).
2. Place them all in the same directory.
3. Open the `index.html` file in any modern web browser to start playing.

Requirements:  
- A modern web browser (Chrome, Firefox, Edge, etc.).
- No additional libraries or frameworks are needed.

Author:  
This game was created using HTML, CSS, and JavaScript. Have fun playing!
